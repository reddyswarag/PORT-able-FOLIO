instructions to implement this code 

way 1 : 
    - download VisualStudio code in you system
    - create a new folder and name it as you wish
    - open the folder in VisualStudio code 
    - in the extensions search for and download live server
    - in the terminal use the following commands in the specified order 
        * git clone https://github.com/reddyswarag/PORT-able-FOLIO.git
        * cd PORT-able-FOLIO
        * then in the explorer panel open the PORT-able-FOLIO folder 
        * right click on the index.html
        * select the first option open with live server
        you can see my portfolio ! welcome :)

way 2 :

    - copy the index.html code from this git hubpage and then past it into a txt editor and save it as file_name.html
    - right click on the file you created and select open with and choose one web browser and youll see my portfolio !
